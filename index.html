How to set Visual Studio Code

To set Visual Studio Code you first go to the website:https://code.visualstudio.com/Download. After it is opened you have to click on the respective iOS you are using. Once it is clicked it will be downloaded. It will open and you should make a folder on the device. After that you should open the folder and you can start coding!

To print something on the console you have first type "console.log()"in the brackets you can write the variable that should be printed. To see if it worked or not you go live and right click on the screen. On the pop-up find the word Inspect and click on it. When this happens a new screen should be shown on the right side. Click on Console and you will find the results of your code on it.
